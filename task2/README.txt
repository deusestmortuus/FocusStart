Параметры программы задаются при запуске через аргументы командной строки:

    - Вывод результата работы программы (-c или -f), обязательный;
    - имя входного файла с названием и параметрами фигуры, обязательный;
    - имя выходного файла, обязательное, если вывод результата осуществляется в файл;

Примеры запуска из командной строки для Windows (для приложенного файла "merge_sort.jar"):

    java java -Dfile.encoding=UTF-8 -jar task2-1.0-SNAPSHOT.jar -f in.txt out.txt
    java java -Dfile.encoding=UTF-8 -jar task2-1.0-SNAPSHOT.jar -c in.txt
